# Changelog

## 0.1.0 – 2026-02-02

- Curse stages for items
- Dedicated Curse Description section on item sheets
- GM-only curse management with player visibility control
- "Curse Effect" flag for Active Effects
- Automatic hiding of unrevealed cursed effects from players
- World settings to:
    - Block unequipping cursed items
    - Block breaking attunement on cursed items
    - Restrict curses to attuned items only
- Localization-ready architecture
